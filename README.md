# 🎬 Ciné Fan

Application React de recherche et découverte de films, propulsée par l'API TMDB.

## Démo

[Ciné Fan en ligne](https://GCLAUDE21.github.io/Cine-fan/)

## Fonctionnalités

- Recherche de films par nom
- Tri des résultats par note (Top / Flop)
- Affichage des détails : affiche, date de sortie, note, genres, synopsis
- Ajout / suppression de films en coups de coeur
- Sauvegarde des favoris en localStorage

## Stack technique

- React + Vite
- React Router DOM
- Axios
- API TMDB
- FontAwesome
- CSS / SCSS
